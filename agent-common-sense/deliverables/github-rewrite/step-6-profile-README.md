The first open-source bridge connecting AI to Autodesk Revit. Full read-write access to BIM models.
- 705+ MCP endpoints | 146 C# files | 113 knowledge files
- 5 levels of autonomy — from direct API calls to goal-driven execution
- AI can create walls, place doors, generate sheets, check code compliance

## The Thesis

The AEC industry runs on Revit. Every architect, every engineer, every contractor touches it. But Revit has no AI integration. No API that AI can talk to natively. No way for an agent to *do* something in a model.

I'm fixing that. One endpoint at a time.

**The stack:**
```
User says "create construction documents"
    → Autonomy Engine decomposes into goals and plans
    → Alignment Core injects domain-specific corrections
    → RevitMCPBridge executes 705+ operations in Revit
    → Corrections from mistakes feed back into future agents
```
