# BridgeAI User Guide

## Getting Started

BridgeAI is your personal computer assistant. Just talk to it like you would a helpful friend.

### Starting BridgeAI

1. Open a terminal (Command Prompt or PowerShell)
2. Type `claude` and press Enter
3. Start talking!

### Basic Commands

You don't need to memorize commands. Just ask naturally:

**Instead of:** Complex computer stuff
**Just say:** What you want to happen

## Safety Features

BridgeAI is designed to be safe:

- **Always asks before deleting** - You'll confirm before anything is removed
- **Explains what it's doing** - No mystery actions
- **Can undo most things** - Mistakes can be fixed
- **Protects important files** - System files are off-limits

## Tips for Best Results

1. **Be specific** - "Print the document on my desktop called report.pdf" works better than "print something"

2. **Ask follow-ups** - If you don't understand something, ask "Can you explain that?"

3. **Say what you want, not how** - "I need to clean up my computer" is better than asking about specific commands

4. **It's okay to make mistakes** - BridgeAI will ask for clarification if needed

## Common Questions

### "Is this safe?"
Yes! BridgeAI won't delete important files or make dangerous changes without your permission.

### "What if I say something wrong?"
No problem - just clarify or say "never mind" and start over.

### "Can it see my passwords?"
BridgeAI can help you manage passwords securely, but it won't share them or access accounts without you asking.

### "What if I need to stop it?"
Say "stop" or press Ctrl+C at any time.

### "Does it work offline?"
BridgeAI needs internet to work - it connects to AI services to understand you.

## Getting More Help

If you're stuck, just ask:
- "Help" - Shows what BridgeAI can do
- "What can you help me with?" - Lists capabilities
- "I don't understand" - Gets a simpler explanation
