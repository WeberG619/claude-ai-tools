# BridgeAI

**Your AI that actually controls your computer.**

You talk. It does.

## What Is This?

BridgeAI is a personal AI assistant that doesn't just chat - it takes action. It can:

- **Hear you** - Just talk naturally
- **Do things** - Create files, print documents, open apps
- **Fix problems** - Diagnose and repair system issues
- **Remember** - Knows your preferences across sessions
- **See your screen** - Take screenshots, read what's displayed

## Quick Start

1. Install Claude Code CLI
2. Run the setup script
3. Start talking

```bash
# From this directory
./scripts/setup.sh

# Then just run
claude
```

## What Can It Do?

### Everyday Tasks
- "Create a document called grocery list"
- "Print the file on my desktop"
- "Open Chrome and search for weather"
- "Take a screenshot"
- "Find all PDFs in my Downloads folder"

### System Help
- "My computer is running slow"
- "What's using all my memory?"
- "Clean up my disk space"
- "Why won't my internet work?"
- "Stop that frozen program"

### Organization
- "Remember that my doctor's name is Smith"
- "What did we talk about yesterday?"
- "Organize my desktop files by type"

## Safety

BridgeAI always:
- Asks before deleting anything
- Explains what it's about to do
- Can undo most actions
- Won't touch system-critical files without explicit permission

## For Non-Technical Users

You don't need to know anything about computers. Just talk to it like you would a helpful friend who happens to be good with technology.
