Act 2

The Architecture

Five Phases. One Engine.

Autonomy Engine — Five-Phase Architecture

1. Goal Engine

Hierarchical goals with weighted progress cascade. Parent goals auto-calculate from children. Persists across sessions.

2. Planner

Decomposes goals into atomic steps. MECE validation. Recursive sub-decomposition. 4 built-in templates.

3. Alignment Core

Compiles per-agent injection prefix. Core principles + domain-aware corrections. Auto-domain detection across 9 domains.

4. Coordinator

Tracks active agents. Resource locks prevent collisions. Context compression between steps. Coherence monitoring.

5. Integration Wiring

PreToolUse hook intercepts every sub-agent dispatch. Alignment Core compiles injection. Dispatcher delivers. Agent runs with accumulated knowledge. No manual copy-paste. Automatic. Every time.
