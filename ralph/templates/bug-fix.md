## Instructions
1. Reproduce the bug to understand it
2. Write a failing test that captures the bug
3. Find the root cause in the code
4. Fix the issue
5. Verify the test now passes
6. Run all tests to ensure no regressions
7. Commit with message explaining the fix

## Investigation Steps
- Check recent commits that might have introduced the bug
- Add logging/debugging to trace the issue
- Review related code for similar patterns

## Quality Standards
- Bug is fixed
- New test prevents regression
- All existing tests pass
- Fix doesn't introduce new issues
